# About Bonfire Terminal

## Boilerplate (for press use)

Bonfire Terminal is a local-first AI platform built by M3M3TIC LLC that gives marketers, content creators, and coaches direct control over artificial intelligence on their own hardware. Unlike cloud-based AI services, Bonfire runs models, speech-to-text, image editing, and video creation entirely on the user's machine — no data leaves the device. Users interact with AI through their existing messaging apps (Telegram, Discord, WhatsApp, Slack) via Bonfire's bridge architecture. The product ships as a single Windows installer with 16 bundled toolchains and requires zero external dependencies or API keys to get started.

## About M3M3TIC LLC

M3M3TIC LLC is a technology company focused on local-first AI tools and creator-economy infrastructure. Founded by John Crestani, the company builds software that prioritizes transparency, privacy, and user control over cloud convenience. Headquarters: United States.

## Product Facts

- **Category**: Desktop AI Application / Content Creation Tool
- **Platform**: Windows 10+ (macOS planned)
- **Price**: $5,000 one-time investment (no subscription, no recurring fees)
- **Architecture**: Rust daemon + Electron UI + Bazel build system
- **Open components**: Build pipeline, quality gates, and toolchain validation are open-source
