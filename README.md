## For Affiliates

1. Download badges from [`badges/`](badges/) — embed with your referral link using the code in `EMBED.md`
2. Grab banners from [`banners/`](banners/) — 6 standard sizes for display ads
3. Clone landing pages from [`pages/`](pages/) — find-replace `{{REFERRAL_URL}}` with your tracking link
4. Use email swipe copy from [`email/swipe-copy/`](email/swipe-copy/)

All assets use `{{REFERRAL_URL}}` as a placeholder — works with any affiliate tracking platform.

## What is Bonfire Terminal?

Bonfire Terminal is a local-first AI platform for content creators. It runs AI models, speech-to-text, and media editing directly on your machine — no cloud, no API keys, no data leaving your device. Interact through your phone's messaging app (Telegram, Discord, WhatsApp, Slack) and get back AI-generated content in seconds.

Learn more at [bonfireterminal.com](https://bonfireterminal.com).
